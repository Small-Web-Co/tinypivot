# TinyPivot — Vue Starter

A minimal [Vite](https://vitejs.dev/) + Vue 3 example showing [TinyPivot](https://tiny-pivot.com) with 50 rows of sample sales data and the pivot table enabled.

[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/github/Small-Web-Co/tinypivot/tree/master/examples/stackblitz-vue)

## What's included

- 50 rows of realistic sales data (region, product, category, sales, units, quarter, year, rep)
- `:show-pivot="true"` — click the pivot icon in the toolbar to explore aggregations
- CSV export, global search, pagination, column resize, clipboard copy
- `theme="light"` — swap to any of TinyPivot's 22 built-in themes

## Run locally

```bash
npm install
npm run dev
```
